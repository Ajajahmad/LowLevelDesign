                                     ** Requirements:**

1️⃣ Who are the users?

**Users can be:**
Guests (read-only)
Registered Users

**Only registered users can:**
Ask questions
Answer questions
Comment
Vote
(No admin/moderator logic needed for now.)

2️⃣ What actions can users take?
**Core actions (IN SCOPE):**
Ask a question
Answer a question
**Comment on:**
Questions
Answers
**Upvote / Downvote:**
Questions
Answers
**Out of scope:**
Editing posts
Closing questions
Flagging
Bounties
Badges

**3️⃣ What is in scope vs out of scope?**
✅ **In Scope:**
Posting questions
Posting answers
Voting
Commenting
Basic reputation system
Accepting an answer

❌ **Out of Scope:**
Search
Tags hierarchy
Notifications
Moderation tools
Pagination
Performance optimization

**4️⃣ Reputation — how should it work?**
Reputation is user-based
Simple rules:
+10 for upvote on question/answer
−2 for downvote
+15 for accepted answer
Reputation updates immediately
